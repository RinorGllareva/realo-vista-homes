# Realo Real Estate Frontend

Public website and admin dashboard for Realo Real Estate.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```

## Deployment

The production frontend is deployed on Vercel and served at:

https://www.realo-realestate.com
